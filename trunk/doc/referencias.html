Referencias

Aima

Hemos utilizado la librería Aima para Java. Se trata de una basta librería de algoritmos de búsqueda. Una vez implementada la clase Puzzle.java ha sido muy sencillo incorporar distintos puzzles a la aplicación, pudiéndonos centrar en la creación de reglas y heurísticas para cada puzzle.

SWT - Standard Widget Toolkit

Para tener un entorno gráfico amigable y similar al del sistema operativo sobre el que se ejecuta la aplicación, hemos utilizado la librería SWT para Eclipse.
